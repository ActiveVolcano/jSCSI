ViSCSI Usage Instructions

ViSCSI is a Eclipse plugin and visualizing tool for jSCSI, which supports the analysis of block layout and access patterns. You only have to install ViSCSI in Eclipse and have access to an jSCSI server via network for using ViSCSI.

Once you have started Eclipse with ViSCSI, open the window "Show View" (Window → Show View → Other...), select ViSCSI/ViSCSI Control Center and press OK. Enter the network path to your jSCSI server and click on the "Apply" button. Click on "New Histogram" or "New Pattern" depending on what you want to see. In the new visible Views select the device you want to display and the options as needed. If you want to have the legend as well, go to the Show View window and select ViSCSI/ViSCSI Legend.

If there are some block-type informations as well, insert the path to the type log-file and press the "Apply " button. Sometimes there are only rare hits on all blocks, to make them more visible, use the Scales in the " ViSCSI Control Center".
