32       * <p>

35       * reliability of the logical unit (e.g., read scan).

42       * is set to one and EWASC is set to zero. If the EWASC bit is set to one,
43       * warning reporting shall be enabled. The method for reporting the warning
44       * when the EWASC bit is set to one is determined from the MRIE field.
45       */
46      private final boolean enableWarning;

50       * failure prediction threshold exceeded reporting shall be enabled. The
51       * method for reporting the failure prediction threshold exceeded when the
52       * DEXCPT bit is set to zero is determined from the MRIE field. A DEXCPT bit
53       * set to one indicates the device server shall disable reporting of the
54       * failure prediction threshold exceeded. The MRIE field is ignored when
55       * DEXCPT is set to one and EWASC is set to zero.
56       */
57      private final boolean disableExceptionControl;

59      /**
60       * A TEST bit set to one shall create a test device failure at the next
61       * interval time, as specified by the INTERVAL TIMER field, if the DEXCPT
62       * bit is set to zero. When the TEST bit is set to one, the MRIE and REPORT
63       * COUNT fields shall apply as if the TEST bit were zero. The test device
64       * failure shall be reported with the additional sense code set to FAILURE
65       * PREDICTION THRESHOLD EXCEEDED (FALSE). If both the TEST bit and the
66       * DEXCPT bit are one, the MODE SELECT command shall be terminated with
67       * CHECK CONDITION status, with the sense key set to ILLEGAL REQUEST, and
68       * the additional sense code set to INVALID FIELD IN PARAMETER LIST. A TEST
69       * bit set to zero shall instruct the device server not to generate any test
70       * device failure notifications.
71       */
72      private final boolean test;

74      /**

87       * vendor specific.

89      private final int methodOfReportingInformationalExceptionConditions;

94       * occurred. The device server shall not report informational exception
95       * conditions more frequently than the time specified by the INTERVAL TIMER
96       * field and shall report them after the time specified by INTERVAL TIMER

98       * reported the interval timer shall be restarted.
99       * <p>
